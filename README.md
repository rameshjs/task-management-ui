# Task Management UI

Frontend for the task management API. Built with React, TypeScript, Vite, shadcn/ui, TanStack Table, and React Query.

## Setup

```
bun install
```

Create a `.env` file:

```
VITE_API_BASE_URL=http://localhost:8080
```

## Run

```
bun run dev
```

## Build

```
bun run build
```
